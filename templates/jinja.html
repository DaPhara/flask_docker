1. String Functions:

{{ "hello world" | length }} # Output: 11
{{ "hello world" | reverse }} # Output: dlrow olleh
{# {{ "hello world" | slice(0, 5) | string }} # Output: hello#}

2. Text Formatting Functions:

{{ "hello world" | capitalize }} --- Output: Hello World
{{ "hello world" | lower }} --- Output: hello world
{{ "hello world" | upper }} --- Output: HELLO WORLD
{{ "hello world" | title }} --- Output: Hello World
{{ " hello world " | trim }} --- Output: hello world

3. Math Functions:

{{ -4 | abs }} # Output: 4
{{ 5 | float }} # Output: 5.0
{{ 5.9 | int }} # Output: 5
{{ 5.444 | round }} # Output: 5

4. List Functions:

{{ [1, 2, 3, 4, 5] | first }} # Output: 1
{{ [1, 2, 3, 4, 5] | last }} # Output: 5
{{ [1, 2, 3, 4, 5] | length }} # Output: 5

5. URL Functions:

{{ "https://example.com" | urlize }} # Output: https://example.com
{{ "https://example.com" | urlencode }} # Output: https%3A%2F%2Fexample.com

6. Miscellaneous Functions:

{{ "hello world " | wordcount }} # Output: 2
{{ "hello world " | wordwrap }} # Output: hello world
